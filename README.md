# meals-dashboard

#Description:
Meals Dashboard is a SPA(Single Page web Application) developed in Vuejs, HTML5, CSS3, Javascript, Bootstrap, Nodejs, Express etc. This dashboard is having 2 views/pages.
1. Home Page - consists of latest meals, popular ingredients and random meals and also having search button to search and see the specific meal you wanted.
2. Meal Page - consists of name of the meal, image of the meal, cooking instructions and a youtube video to go through it.

## Project setup
```
npm install
```
